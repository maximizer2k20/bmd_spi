These tests were developed for BDMware 3.2.0

They rely on teraterm v4.9.4 which can be downloaded from here: https://osdn.net/projects/ttssh2/releases/

The scripts are written in teraterm macro languge. Execute them by running ttpmacro (teraterm pro macro) located in C:\Program Files (x86)\teraterm.

Results must be analyzed by hand.
